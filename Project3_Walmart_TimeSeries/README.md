## Project 3 — Walmart Sales Analysis

This project analyzes Walmart sales data using time series and business analysis techniques.

### Objectives
- Analyze sales trends over time
- Evaluate store performance
- Study the effect of holidays and external factors
- Apply forecasting techniques

### Steps Performed
- Converted date column to datetime format
- Sorted data by time
- Created time-based features (month, year)
- Analyzed sales trends over time
- Compared store performance
- Evaluated holiday impact on sales
- Applied ARIMA model for forecasting

### Key Insights
- Sales show clear trends and seasonal patterns
- Holiday periods increase sales performance
- Some stores significantly outperform others
- Time-based features are important for analysis
- Forecasting provides useful future estimates

### Tools Used
- Python
- Pandas
- Matplotlib
- Statsmodels
