## Project 1 — Data Analysis

This project focuses on exploratory data analysis (EDA) using a structured dataset.

### Objectives
- Understand the dataset structure
- Clean and preprocess the data
- Identify patterns, correlations, and outliers

### Steps Performed
- Removed non-numerical columns
- Handled missing values using mean imputation
- Removed duplicate records
- Performed statistical analysis
- Visualized distributions and correlations

### Key Insights
- Features show different distributions and scales
- Some variables are skewed and not normally distributed
- Correlation analysis highlights relationships between variables
- Outliers are present and may affect further modeling

### Tools Used
- Python
- Pandas
- Matplotlib
