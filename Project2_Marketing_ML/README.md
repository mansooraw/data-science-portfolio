## Project 2 — Marketing Campaign Prediction

This project uses machine learning to predict customer response to marketing campaigns.

### Objectives
- Prepare data for modeling
- Build a predictive model
- Evaluate model performance

### Steps Performed
- Selected numerical features
- Handled missing values using median imputation
- Applied feature scaling
- Split data into training and testing sets
- Trained a Random Forest classifier
- Evaluated model accuracy
- Analyzed feature importance

### Key Insights
- The model achieved good prediction performance
- Some features have higher influence on customer response
- Machine learning can support targeted marketing strategies

### Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
